OPERATING GUIDE

用 PaperLens 快速建立论文版图

它适合用来做技术预研、竞品路线梳理、方案立项前的文献扫描。流程很短：配置模型，输入需求，等待检索和精读，最后查看按路线分组的报告。

01

配置模型

  • 至少填写一个模型服务的 API Key。
  • 检索、精读、综述三个档位可以分别选择不同模型。
  • 精读档位最影响单篇论文提炼质量。
02

输入研究 brief

  • 写清研究对象、技术方向、约束条件和评估指标。
  • 论文数量越多，运行时间和模型费用越高。
  • 时间范围用于限制近期论文。
03

阅读结构化报告

  • 报告按技术路线自动分组。
  • 核心论文显示全文精读结论，长尾论文显示摘要级研判。
  • 离线 HTML 报告会保存到 reports 目录。
04

调整下载源

默认使用 arxiv.org。国内网络下载 PDF 慢时，可以在设置里使用中科院理论物理所历史镜像地址：

http://xxx.itp.ac.cn

提示：arXiv 镜像只影响 PDF 下载，不影响论文检索。若镜像不可用，程序会回退到官方 PDF 地址。
